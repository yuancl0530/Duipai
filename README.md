# 对拍  
## UNIX版  
### 使用方法  
1.把标准程序拷贝到`std.cpp`文件中;  
2.把测试程序拷贝到`test.cpp`文件中;  
3.根据需要修改随机数生成程序`makedata.cpp`  
4.make  

## WINDOWS版  
### 使用方法  
1.把标准程序拷贝到`std.cpp`文件中;  
2.把测试程序拷贝到`test.cpp`文件中;  
3.根据需要修改随机数生成程序`makedata.cpp`  
4.编译运行`check.cpp`文件  
## 说明
>1.check.cpp中Time表示测试数据数量  
2.出现错误的测试数据以及对应的正确运行结果和错误的运行结果将保存在WrongData文件中
